# CollectionsWithTelusko

Java Collections practice based on Telusko tutorials. This repository contains hands-on examples and exercises that demonstrate core Java Collections Framework concepts such as `List`, `Set`, `Map`, sorting and comparisons.

## Getting Started

Prerequisites
- Java Development Kit (JDK) 11 or newer
- `IntelliJ IDEA 2025.3` (recommended) or another Java IDE
- Git (to clone the repository)

Clone the repository:
- `git clone https://github.com/omarAbdullahMoharam/CollectionsWithTelusko.git`
- `cd CollectionsWithTelusko`

Open the project in `IntelliJ IDEA 2025.3` and run example classes directly using the IDE run configurations.

Command-line (Windows) quick run
- Compile: `javac -d out src\*.java`
- Run an example (replace `Main` with the desired class name): `java -cp out Main`

Examples you can run:
- `java -cp out Main`
- `java -cp out ComparatorAndComparable`
- `java -cp out SortingCollections`

If your sources are inside packages, adjust the `javac` and `java` commands to use package-qualified class names and paths.
